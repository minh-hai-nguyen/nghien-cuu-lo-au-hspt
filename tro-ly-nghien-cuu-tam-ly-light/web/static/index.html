Hướng dẫn sử dụng

1. Giới thiệu

Trợ lý Nghiên cứu Tâm lý học là công cụ web hỗ trợ nghiên cứu về sức khỏe tâm thần ở học sinh. Sử dụng công nghệ RAG (Retrieval-Augmented Generation) để tìm kiếm trong 1.907 đoạn văn từ 54+ tài liệu. Hỗ trợ tiếng Việt và tiếng Anh. Không cần đăng nhập.

2. Tab Hỏi đáp

Nhập câu hỏi → hệ thống trả về các đoạn liên quan nhất từ bài báo. Mỗi kết quả có:

  • Badge xanh "82%" = độ liên quan
  • Badge cam "Trang 5" = vị trí trong bài gốc
  • Bấm tiêu đề = mở PDF bản dịch
  • "Xem bản gốc" = PDF tiếng Anh từ tạp chí

3. Tab Tổng hợp liên bài

6 bảng so sánh kết quả từ nhiều bài báo: tỷ lệ theo quốc gia, sàng lọc vs chẩn đoán, yếu tố nguy cơ/bảo vệ, xu hướng thời gian, can thiệp, dịch vụ SKTT. Bấm link trong bảng → mở PDF.

4. Tab Thư viện

28 file PDF + DOCX (bản dịch, bản gốc, báo cáo). Bấm xem hoặc tải.
